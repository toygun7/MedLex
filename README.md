# 🩺 MedLex

MedLex is a web-based application designed to help medical students learn English medical vocabulary.

## Features

- 📚 Medical vocabulary categorized by topic
- ❤️ Favorite words
- 🔍 Category filtering
- 📝 Interactive quiz mode
- 🔄 English ↔ Turkish quiz
- 📖 Example sentences
- 💾 Local storage support

## Categories

- General Medicine
- Anatomy
- Diseases
- Medical Verbs

## Technologies

- HTML5
- CSS3
- JavaScript
